Static hasAllBits

  • hasAllBits(param0: Object): boolean
    • Defined in gibbon-processor.ts:102

    Checks if all logical "1" bits from byte2 compare to the ones in byte1

    example
    // 1. 0101 0010       (82 dec)
    // 2. 0100 0010       (66 dec)
    // ------------- AND
    //    0100 0010       (66 dec)
    //    66 dec === 66 dec :-)
    //

    // 1. 0010 1100       (44 dec)
    // 2. 0010 1000       (40 dec)
    // ------------- AND
    //    0010 1000       (40 dec)
    //    40 dec === 40 dec :-)

    // 1. 0000 0000       (0 dec)
    // 2. 0001 0000       (16 dec)
    // -------------- AND
    //    0000 0000       (0 dec)
    //     0 dec !== 16 dec :-(


    // 1. 0000 0000       (0 dec)
    // 2. 0010 0010       (34 dec)
    // -------------- AND
    //    0000 0000       (0 dec)
    //     0 dec !== 34 dec :-(

    Parameters

    • param0: Object

    Returns boolean

Static hasAnyBits

  • hasAnyBits(param0: Object): boolean
    • Defined in gibbon-processor.ts:63

    Compare 2 bytes, matches when byte1 has any similar bits set compared to byte2

    example
    // 1. 1000 1001
    // 2. 0000 1000
    // ------------- AND
    //    0000 1000       (8 dec)
    //      (8 dec !== 0x0) => true

    const byte1 = 0x89; // 137 (dec)
    const byte2 = 0x8; // 8 (dec)

    hasAnyBits({byte1, byte2}); // returns true

    Returns boolean
